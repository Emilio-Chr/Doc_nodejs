"Hola Mundo" con Express y Node.js

Inicializar un proyecto con NPM

sh


npm init -y

Instalar Express.

sh


npm install express

Crear el archivo del servidor

js


const express = require('express'); // Importar Express
const app = express(); // Crear una aplicación Express
                
const PORT = 3000; // Puerto donde correrá el servidor
                
// Ruta principal que devuelve "Hola Mundo"
app.get('/', (req, res) => {
    res.send('¡Hola Mundo!');
});
                
// Iniciar el servidor
app.listen(PORT, () => {
    console.log(`Servidor corriendo en http://localhost:${PORT}`);
});

Ejecutar el servidor.

sh


node index.js
