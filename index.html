¿Que es Node.js?

Node.js es un entorno de ejecución de JavaScript que se utiliza para crear aplicaciones web desde el servidor. Es una plataforma multiplataforma, de código abierto y gratuita.

Historia de Node.js

Historia de Node.js

2009

La historia de Node.js comienza en 2009 cuando su autor, Ryan Dahl, presentó por primera vez esta plataforma en la conferencia JSConf EU.

27 de mayo de 2009

Dahl presentó oficialmente Node.js como un entorno de tiempo de ejecución de JavaScript de código abierto en el lado del servidor.

Problema a resolver

Buscaba una forma de crear servidores web en tiempo real altamente escalables, ya que Apache HTTP Server tenía problemas para manejar muchas conexiones simultáneas.

Plataformas iniciales

Inicialmente, Node.js solo funcionaba en Linux y Mac OS X.
